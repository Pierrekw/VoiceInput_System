FunASR CPU优化版本 - 快速部署指南
=================================

🚀 三步快速开始：

1️⃣ 运行安装脚本
   Windows: 双击 setup_windows.bat
   Linux:   ./setup_linux.sh

2️⃣ 复制模型文件
   将模型文件复制到 model\fun\ 目录
   必需文件：model.pt, config.yaml, tokens.json, am.mvn

3️⃣ 测试运行
   python test_cpu_basic.py      # 基础测试
   python test_funasr_cpu.py     # 语音识别

❓ 常见问题：
- Python未找到：重新安装Python并勾选Add to PATH
- PyAudio失败：Windows需安装Visual C++ Build Tools
- 麦克风问题：检查系统音频权限设置

📋 系统要求：
- Windows 10/11 或 Linux
- Python 3.8+
- 4GB+ 内存
- 麦克风设备

📖 详细说明请查看：
- README.md - 完整说明文档
- deployment_guide.md - 详细部署指南
- ONNX_OPTIONAL.md - ONNX可选优化（高级用户）

🚀 高级选项（可选）：
如果需要极致性能，可以尝试ONNX版本：
1. 查看 ONNX_OPTIONAL.md
2. 运行 optional_onnx_deps.bat
3. 运行 optional_build_onnx.bat
