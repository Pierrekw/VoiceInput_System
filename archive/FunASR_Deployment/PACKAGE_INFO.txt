🚀 使用流程：

1️⃣ 解压部署包到目标文件夹
2️⃣ 运行对应系统的安装脚本
3️⃣ 复制模型文件到 model/fun/ 目录
4️⃣ 运行基础测试验证安装
5️⃣ 开始使用语音识别功能

📋 模型文件需求：
必须从原始环境复制以下文件到 model/fun/：
- model.pt          # 约1-2GB，模型权重
- config.yaml       # 模型配置
- tokens.json       # 词汇表
- am.mvn           # 音频参数
- configuration.json # 可选，元信息

⚡ 性能特点：
- CPU推理优化，适合无GPU环境
- 低内存占用（1-2GB）
- 实时语音识别延迟0.5-1秒
- 支持VAD语音活动检测
- 集成标点符号恢复

🛠️ 系统要求：
- Windows 10/11 或主流Linux发行版
- Python 3.8 或更高版本
- 4GB+ 系统内存
- 麦克风设备
- 2GB+ 可用磁盘空间
